# Pearagon Task 2

Task 2 is a TypeScript Express server that serves a small vanilla frontend from the `public/` directory.

## Install

`npm install`

## Run

`npm start`

## Development

`npm run dev`

## Current setup

The server compiles TypeScript into `dist/`, serves static files from `public/`, and returns `index.html` at `/`.

Count routes and button behavior are intentionally not implemented yet in this setup step.
